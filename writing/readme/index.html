Writing Assets

`Resources/Writing` 是 KB 的写作资产层，保存可以跨项目复用的写作规范、结构模板、质量清单、样例和正式文档归档入口。

Resources/Writing 是 KB 的写作资产层，保存可以跨项目复用的写作规范、结构模板、质量清单、样例和正式文档归档入口。

Boundary

KB 负责保存长期资产：

  • 风格规范
  • 文档模板
  • 质量清单
  • 优秀样例和反例
  • 可复用 prompt / playbook
  • 已确认具备长期价值的正式文档

CLI doc-writer 负责执行：

  • 从 KB 读取写作资产
  • 读取当前项目上下文
  • 在当前项目生成、改写、评审文档
  • 把通用沉淀提议回 KB

Directory

  • Styles/：风格包。
  • Templates/：结构模板。
  • Checklists/：质量检查清单。
  • Examples/：优秀样例和反例入口。
  • Docs/：长期正式文档归档入口。

Default Flow


其他项目请求写作资产
  -> KB 返回风格、模板、示例、checklist
  -> CLI 在当前项目生成文档
  -> 当前项目保存文档
  -> 只有通用资产经确认后回传 KB

默认不要把每篇业务文档都回传 KB。KB 只接收可复用、已脱敏、长期有价值的资产。
