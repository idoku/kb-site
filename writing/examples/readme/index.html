Rules

  • 只收录可复用、已脱敏、长期有价值的文档。
  • 优秀样例要说明为什么好。
  • 反例要说明风险，不保留敏感细节。
  • 普通业务项目文档不要默认收录。

Candidate Types

  • 技术设计文档
  • 产品方案
  • PRD
  • RFC / ADR
  • 复盘
  • 周报
  • 迁移方案
  • 发布说明

Contribution Flow


当前项目完成文档

  -> 人确认
  -> 写入 Examples 或 Templates
